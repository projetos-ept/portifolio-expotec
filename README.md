# Portifólio Expotec

Bem-vindo ao repositório "portifolio-expotec". Este projeto contém um portfólio em HTML, criado para apresentar projetos e informações pessoais/academicas de forma simples e estática.

## Visão geral
- Linguagem principal: HTML (100%)
- Tipo: Site estático (uma página ou conjunto de páginas HTML)
- Uso: Apresentação de projetos, habilidades e contato

## Estrutura do projeto
A estrutura típica do repositório é:
- index.html — página principal do portifólio
- /assets (opcional) — imagens, ícones e outros recursos
- /css (opcional) — arquivos CSS
- /js (opcional) — scripts JavaScript

(Verifique o conteúdo do repositório para a estrutura exata. Caso possua subdiretórios, adapte conforme necessário.)

## Como executar localmente
Como este é um site estático em HTML, você pode abri-lo diretamente no navegador ou servir localmente:

Opção 1 — Abrir diretamente
1. Faça o download/clonagem do repositório.
2. Abra o arquivo `index.html` no seu navegador (duplo clique ou “Abrir arquivo”).

Opção 2 — Servir com Python (recomendado para evitar problemas com rotas/recursos)
1. Navegue até a pasta do projeto no terminal.
2. Execute:
   - Python 3: `python3 -m http.server 8000`
3. Abra `http://localhost:8000` no navegador.

Opção 3 — Usar extensão (VS Code)
- Instale a extensão "Live Server" e clique em "Go Live" para visualizar automaticamente em `http://127.0.0.1:5500` (ou porta configurada).

## Personalização
- Edite `index.html` para alterar texto, títulos, seções e links.
- Adicione/edite arquivos CSS em `/css` para mudar estilo e responsividade.
- Coloque imagens em `/assets` e referencie-as no HTML.

## Boas práticas recomendadas
- Mantenha o HTML semanticamente estruturado (header, main, section, footer).
- Use etiquetas de acessibilidade (alt em imagens, roles quando necessário).
- Teste em diferentes tamanhos de tela para garantir responsividade.
